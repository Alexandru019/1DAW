    -----CONDICIONALES DE CADENAS-----
    
==  Compara si son iguales las cadenas 
-z  Comprueba que la cadena está vacia
-n  Comprueba que la cadena no está vacia
!=  Devuelve true si las cadenas no son iguales

    -----CONDICIONALES DE FICHEROS-----

-f <ruta del fichero>       Comprueba que es un fichero,devuelve true
-d <ruta del directorio>       Comprueba que es un directorio,devuelve true
-s <ruta del fichero>       Comprueba que el fichero no está vacío,devuelve true
-r <ruta del fichero o directorio>      Comprueba el permiso de lectura
-w <ruta del fichero o directorio>      Comprueba el permiso de escritura
-x <ruta del fichero o directorio>      Comprueba el permiso de ejecucion
